=============================================================================
DENODO MCP SERVER
=============================================================================

Denodo MCP Sever allows users to connect to the Denodo Platform and query
its databases acting as a Model Context Protocol server interface
(see https://modelcontextprotocol.io/specification/2025-06-18).

For running the Denodo MCP Server, you need to execute the denodo-mcp-server.bat/.sh script.

This software is part of the DenodoConnect component collection.
